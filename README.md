# Convert MicroSoft Office Word files to PDF ![v](https://em-content.zobj.net/source/samsung/395/ping-pong_1f3d3.png)
## Super light powershell script to convert a whole folder to PDF files by one click. :)
**Download latest version of** [powershell-Word2PDF](https://github.com/2muu/powershell-Word2PDF/releases/download/v2026.4.0/powershell-Word2PDF.ps1)
* Windows 8,10,11 with MS Office word installed;
* Right click powershell-Word2PDF.ps1 run with powershell;
* Choose folder which contains source files;
* Click OK;
* Check source folder.
